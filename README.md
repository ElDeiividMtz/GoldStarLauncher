# GoldStar Launcher

Launcher oficial de GoldStar Studios para Minecraft.

## Descarga

Ve a [Releases](https://github.com/ElDeiividMtz/GoldStarLauncher/releases) para descargar la ultima version.

## Instalacion

1. Descarga el `.zip` de la ultima release
2. Extrae en cualquier carpeta
3. Ejecuta `GoldStar Launcher.exe`
